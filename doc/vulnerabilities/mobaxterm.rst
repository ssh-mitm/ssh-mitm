:fas:`desktop` MobaXterm
========================

MobaXterm is a Windows SSH client and terminal emulator that bundles an X11 server, tabbed SSH sessions,
network tools (FTP, SFTP, RDP, VNC, Telnet), and a local Unix environment based on Cygwin.
Its SSH implementation is based on PuTTY.

MobaXterm is developed and maintained by Mobatek and is widely used in enterprise environments.
